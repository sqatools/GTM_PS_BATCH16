1. Install pytest using below command
   ->  pip install pytest

2. Create test file with test_ prefix  e.g. test_smoke_case.py

3. Create test function with test_ prefix in function name e.g.  test_addition(), test_subtractions().

4. execute the test cases with help of command
   ->  python -m pytest -v test_file_name


5. Execute the test cases with specific marker 
  -> python -m pytest -v -m "smoke" .\TestFiles\test_smoke_cases.py

   # and condition with marker
   ->  python -m pytest -v -m "smoke and sanity" .\TestFiles\test_smoke_cases.py

   ->  python -m pytest -v -m "smoke or sanity" .\TestFiles\test_smoke_cases.py



6. Generate HTML Report
   -> install pytest package using below command
   -> pip install pytest-html  

7. Command to Execute test cases with html report on local directory.
  ->  python -m pytest -v .\TestFiles\test_smoke_cases.py --html=report.html 
  ->  It will create report.html file in local directory, that we can open in browser

8. Command to execute specific test cases with name
  -> python -m pytest -v -k "test_smoke_case5" .\TestFiles\test_smoke_cases.py --html=report.html
